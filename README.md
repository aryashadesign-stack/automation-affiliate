# Automation Affiliate (n8n on Render)

Ready-to-deploy configuration for running n8n on Render free plan.

## Steps
1. Create a new GitHub repository (public).
2. Upload these four files to it.
3. Go to Render → Create New → Web Service → Connect your GitHub repo.
4. Wait for build to finish, then open your Render URL.
5. Default login:
   - User: admin
   - Password: admin123
